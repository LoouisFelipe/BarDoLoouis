# 🍻 BarDoLoouis - Sistema de Gestão

Este é o repositório oficial para o sistema de gestão do **Bar do Luis**.

## 🚀 Sobre o Projeto

O projeto **BarDoLoouis** é uma aplicação web moderna construída com **Next.js** e **Firebase** para otimizar as operações diárias do bar, desde o gerenciamento de comandas até a análise de dados para tomada de decisão estratégica.

A aplicação serve como a interface principal para o nosso **Data Hub Unificado**, centralizando informações e fornecendo ferramentas para toda a equipe de gestão.

## 🛠️ Configuração do Ambiente

Para rodar o projeto localmente, você precisa configurar as variáveis de ambiente.

1.  Crie um arquivo chamado `.env.local` na raiz do projeto.
2.  Copie o conteúdo do arquivo `.env.example` (se existir) ou peça as chaves do Firebase para o gestor do projeto.
3.  Preencha as variáveis com as credenciais corretas do ambiente de desenvolvimento.

> **Atenção:** O arquivo `.env.local` contém segredos e **não deve** ser versionado no Git.
